# וייב קודינג ב-Google AI Studio — המדריך לאנשים רגילים

מדריך אינטראקטיבי בעברית (RTL), צעד-אחר-צעד, שמלמד **משתמש חסר כל ידע טכני** איך לבנות אפליקציה אמיתית ב-Google AI Studio — בלי לכתוב קוד ובלי כרטיס אשראי.

נכתב על ידי **תום הגלעדי** עבור האנשים שהוא מלמד את הפלטפורמה.

## מה יש במדריך
- מה זה "וייב קודינג" ומה זה Google AI Studio (בעברית פשוטה, עם אנלוגיות).
- הבדל בין Gemini / AI Studio / Gemini API / Vertex AI — איזה כלי לבחור.
- **מסלול מהיר** למעלה + 11 פרקים מעמיקים מתחת (מבנה דו-שכבתי).
- "האפליקציה הראשונה ב-10 דקות" עם צ'קליסט אינטראקטיבי (נשמר בדפדפן).
- פרק ברור על **חינם מול תשלום** — מתי בכלל ייתכן חיוב, עם תג ₪0 עוקב.
- ארגז פרומפטים מוכנים להעתקה, מילון מונחים, ושאלות נפוצות.

## מבנה טכני
- קובץ HTML יחיד (`index.html`), ללא build step, ללא תלויות (פרט ל-Google Fonts).
- מהלך עיצוב מייחד: רכיב "מסך חי" שמחקה את מסך ה-Build של AI Studio (קלט → פלט).
- גופנים: Frank Ruhl Libre + Assistant. פלטה: קרם / טורקיז / אלמוגי, עם מצב כהה.
- נגישות מלאה (WCAG AA / ת"י 5568): skip-link, ניווט מקלדת, `aria-live`, `prefers-reduced-motion`.
- **נגד התיישנות:** מחירים, מכסות ושמות דגמים אינם מקודדים קשיח — מוצגים כקישור חי לדפי גוגל הרשמיים.

## פיתוח מקומי
פתחו את `index.html` בדפדפן, או הריצו שרת מקומי:
```bash
python -m http.server 8791
```
